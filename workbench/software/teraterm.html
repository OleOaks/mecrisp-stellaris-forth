Tera Term

What’s this used for?

The normal way of communcating with Forth is through a VT100 serial terminal emulator, one example being the free open-source Tera Term.

Forth provides a command line through USART which is typically connected through an inexpensive USART to USB serial converter like a CP2102 USB to UART board. The CP2102 plugs into a usb port on the computer with which Term Term communicates.

Where to get it?

Tera Term is freely downloaded from Tera Term homepage.

Setup Tera Term Quicksteps

  1. Plug the CP2102 USB to UART board into computer.

  2. Open the Tera Term application. (It defaults to a TCP/IP connection)

  3. Select the ‘Serial’ connection option, and click ‘OK’. (a COM port should be displayed in the ‘Port:’ textbox).

  4. Goto ‘Setup’ –> ‘Serial’ dialog.

  5. Change ‘Speed’ to 115200, or whatever your Forth installation will default to. Then click ‘New setting’ to dismiss the dialog box.

  6. Hit the ‘Enter’ key on your keyboard. You should see an ‘ok.’ response from Forth!

  7. Goto ‘Setup’ –> ‘Terminal’ dialog.

  8. Change the New-line Receive value from ‘CR’ to ‘LF’, then click ‘OK’ to dismiss the dialog box.

  9. Goto ‘Setup’ –> ‘Font’ – ‘Font’ dialog.

  10. I change the font to ‘Courier New’ and the size to ‘12’, then click ‘OK’ to dismiss the dialog box.

  11. The default Tera Term window size is a bit small, stretch it to a comfortable size.

  12. Goto ‘Setup’ –> ‘Save setup…’ and create a filename to remember these settings.

  13. The next time you open Tera Term, goto ‘Setup’ –> ‘Restore setup…’ and find your saved settings.

Press the reset button on the microcrontroller, a startup message should be displayed in Tera Term…
